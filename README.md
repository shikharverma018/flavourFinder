🍽️ FlavorFinder – Smart Recipe Explorer

A web app to explore delicious recipes using TheMealDB API.


🎯 Objectives
Integrate a public API using fetch()
Implement search, filtering, and sorting using array higher-order functions
Build a responsive and user-friendly UI
Manage and display dynamic data effectively

🌐 API Used:
TheMealDB API
🔗 https://www.themealdb.com/api.php

Data Retrieved:
Meal name
Meal image
Category (e.g., Dessert, Vegetarian)
Area (e.g., Indian, Italian)
Ingredients
Cooking instructions


🚀 Features
🔍 Search Recipes by name
🎯 Filter Recipes by category or area
🔃 Sort Recipes alphabetically
📖 View Recipe Details (ingredients & instructions)
📱 Responsive Design (mobile, tablet, desktop)
⏳ Loading Indicator while fetching data
❌ Error Handling for failed API requests


⚙️ Technologies Used:
HTML
CSS
JavaScript (ES6)
Fetch API


🧠 JavaScript Concepts Used
fetch() for API calls
Array Higher-Order Functions:
map() → rendering UI
filter() → search & filtering
sort() → sorting results
Event handling
DOM manipulation
